## 安装基本
```
cd /www/server/mdserver-web/plugins && rm -rf mtproxy  && git clone https://github.com/mw-plugin/mtproxy && cd mtproxy && rm -rf .git && cd /www/server/mdserver-web/plugins/mtproxy && bash install.sh install 1.0
```
